Alignment

Default Configuration

Without applying anything else to it, this is the default config. We know what will happen if we apply flex: 1; it will fill the remaining horizontal space. This is shown with the container most bottom. We added flex: auto; to the direct .item container resulting in it filling up all the possible vertical space.

Aligning Items

justify-content:

Justify content aligns items on the main axis, we will show you two examples for now: space-between & center.

Differentiating justify-content and align-items

align-items:

  • Aligns items on the cross-section by default. (row)
  • When applied to an item with flex-direction: column, will align on the main-axis

justify-content:

  • Aligns items on the main-axis by default. (row)
  • When applied to an item with flex-direction: column, will align on the cross-section

Gap

Applies a gap property between all the items. Super useful. The example has a justify-content: center and a gap: 1em.
