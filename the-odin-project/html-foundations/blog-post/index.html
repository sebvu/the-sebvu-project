Installing Arch Linux

So you're interesting in installing Arch Linux?

And you want to manually install it?

Then you've come to the right place! Screw install scripts. We're doing it manual labor

Pre-Requisites

We will be dual booting on our system, so here are the pre-requisites.

This blog post is assumimg you are operating on Windows 11.

  • x86_64 (i.e. 64 bit) compatible machine
  • Recommended 2 GB RAM, minimum of 512 GB
  • Recommended 20 GB disk space, minimum 800 MB
  • 8GB+ Pendrive

If you have all of these, lets get started!

Installing Arch Linux

Creating Partitions

Open windows search bar, and type diskmgmt to access disk manager where you can view all connected drives and their respective partitions.

From here, you want to look for your drive of choice (typically (C: D: E:, or maybe under another alias) and shrink a minimum of 20GB of space for Arch Linux.

This will create an unallocated free space. Now we have free space for installing Arch Linux!
